# Personal Webpage Project

##  Overview
This project is a simple personal webpage built using HTML and CSS. It demonstrates fundamental web development concepts including page structure, styling, and layout.

##  Features
- Structured webpage using HTML (headings, paragraphs, links, images)
- Styling using CSS (colors, fonts, spacing)
- Use of margin and padding for layout
- Reusable class selectors for consistent design
- Basic centered layout using CSS

##  Technologies Used
- HTML5
- CSS3

##  What I Learned
- Difference between HTML structure and CSS styling
- Margin vs padding (outside vs inside spacing)
- Class vs ID selectors
- Importance of readability and color contrast
